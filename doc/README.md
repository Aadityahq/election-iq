# ElectionIQ Project Docs

This folder collects the working context for the project so it is easy to hand off, review, and present during the hackathon.

Files:
- [context.md](context.md) - project goal, problem statement, solution, stack, and architecture
- [todo.md](todo.md) - current implementation status and remaining work
- [logs.md](logs.md) - short development history and milestone notes
- [bugs.md](bugs.md) - known issues, fix plans, and resolved items
